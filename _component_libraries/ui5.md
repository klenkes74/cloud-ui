### Ui5 Web Components

Bundles Ui5 Web Components for use with CloudUi

Current Version: 0.1.0-SNAPSHOT

Maven:
~~~~

<dependency>
    <groupId>net.moewes</groupId>
    <artifactId>cloud-ui-ui5</artifactId>
    <version>0.1.0-SNAPSHOT</version>
</dependency>
~~~~
